# doctoscrap
Data scraper of Doctolib

## 🏁 Installation
```bash
npm install
```

## 🎈 Usage
Set up the `config.js` file

#### Sample config

```javascript
module.exports = {
  query: {
    type: 'medecin-generaliste',
    location: 'rennes'
  }
}
```

#### Run the scraper
```bash
npm start
```
